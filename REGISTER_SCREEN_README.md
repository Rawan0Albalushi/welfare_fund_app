# شاشة إنشاء حساب جديد (Register Screen)

## نظرة عامة

تم إنشاء شاشة تسجيل جديدة بسيطة ومخصصة للتطبيق، تختلف عن شاشة تسجيل الطالب المعقدة. هذه الشاشة مخصصة للمستخدمين العاديين الذين يريدون إنشاء حساب بسيط للمساهمة في التبرعات.

## الميزات الرئيسية

### 1. **معلومات الحساب المطلوبة:**
- **رقم الهاتف** (إجباري) - مع التحقق من صحة التنسيق
- **البريد الإلكتروني** (اختياري) - مع التحقق من صحة التنسيق إذا تم إدخاله
- **كلمة المرور** (إجباري) - مع إمكانية إظهار/إخفاء النص
- **تأكيد كلمة المرور** (إجباري) - مع التحقق من التطابق

### 2. **التحقق من صحة البيانات:**
- التحقق من إدخال رقم الهاتف (10 أرقام على الأقل)
- التحقق من صحة تنسيق البريد الإلكتروني (إذا تم إدخاله)
- التحقق من طول كلمة المرور (6 أحرف على الأقل)
- التحقق من تطابق كلمة المرور مع التأكيد
- التحقق من الموافقة على الشروط والأحكام

### 3. **واجهة المستخدم:**
- تصميم عصري ومتناسق مع باقي التطبيق
- تأثيرات حركية سلسة
- رسائل خطأ واضحة ومفيدة
- مؤشر تحميل أثناء إنشاء الحساب

## التصميم

### 1. **App Bar:**
- خلفية متدرجة جذابة
- أنماط دائرية في الخلفية
- عنوان "إنشاء حساب جديد"

### 2. **قسم الترحيب:**
- أيقونة إضافة شخص مع تصميم متدرج
- رسالة ترحيب "انضم إلينا اليوم"
- وصف "أنشئ حسابك وساهم في مساعدة الطلاب"

### 3. **نموذج التسجيل:**
- حقول إدخال أنيقة مع أيقونات
- تمييز الحقول الاختيارية
- أزرار إظهار/إخفاء لكلمات المرور
- مربع اختيار للشروط والأحكام

### 4. **أزرار الإجراءات:**
- **زر إنشاء الحساب**: مع تصميم متدرج ومؤشر تحميل
- **زر تسجيل الدخول**: للانتقال لشاشة تسجيل الدخول

## التحقق من صحة البيانات

### 1. **رقم الهاتف:**
```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return 'يرجى إدخال رقم الهاتف';
  }
  if (value.length < 10) {
    return 'رقم الهاتف يجب أن يكون 10 أرقام على الأقل';
  }
  return null;
},
```

### 2. **البريد الإلكتروني (اختياري):**
```dart
validator: (value) {
  if (value != null && value.isNotEmpty) {
    if (!RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$')
        .hasMatch(value)) {
      return 'يرجى إدخال بريد إلكتروني صحيح';
    }
  }
  return null;
},
```

### 3. **كلمة المرور:**
```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return 'يرجى إدخال كلمة المرور';
  }
  if (value.length < 6) {
    return 'كلمة المرور يجب أن تكون 6 أحرف على الأقل';
  }
  return null;
},
```

### 4. **تأكيد كلمة المرور:**
```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return 'يرجى تأكيد كلمة المرور';
  }
  if (value != _passwordController.text) {
    return 'كلمة المرور غير متطابقة';
  }
  return null;
},
```

## الفرق بين شاشة التسجيل وشاشة تسجيل الطالب

| الميزة | شاشة التسجيل الجديدة | شاشة تسجيل الطالب |
|--------|---------------------|-------------------|
| **الغرض** | تسجيل عام للمستخدمين | تسجيل خاص للطلاب |
| **المعلومات المطلوبة** | رقم هاتف، إيميل (اختياري)، كلمة مرور | معلومات شخصية وأكاديمية شاملة |
| **التعقيد** | بسيطة وسريعة | معقدة ومفصلة |
| **الاستخدام** | للمساهمة في التبرعات | للتسجيل في صندوق الرفاهية |

## التطوير المستقبلي

### إمكانيات التطوير:
1. **التحقق من رقم الهاتف**: إرسال رمز تحقق عبر SMS
2. **التسجيل الاجتماعي**: تسجيل الدخول عبر Google/Facebook
3. **التحقق من البريد الإلكتروني**: إرسال رابط تأكيد
4. **قوة كلمة المرور**: مؤشر لقوة كلمة المرور
5. **التسجيل السريع**: تسجيل بالبصمة أو الوجه

### التحسينات المقترحة:
1. **الأمان**: تشفير البيانات وإضافة طبقات أمان إضافية
2. **الأداء**: تحسين سرعة التحميل والاستجابة
3. **التجربة**: إضافة خطوات إضافية للتأكيد
4. **الدعم**: إضافة مساعدة مباشرة للمستخدمين
